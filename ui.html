Color migration helper

Run the migration in three steps.

Prime variables

Renames the 'Main color' variable collection to 'Color' and removes the 'color/main/' prefix from variable names.

Clean up variants

Removes unsupported color variants (neutral, support, danger, info, warning, success), updates remaining variant names, and migrates Alert and ValidationMessage to use color modes.

Fix missing instances

Finds instances whose old color variant was removed and swaps them to the matching current variant.
